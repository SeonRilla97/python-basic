# 컨테이너 타입

- 표로 변경

| 타입     | 순서 | 중복 | 수정 | 생성          | 조회                         | 추가/수정            | 삭제                    |
| -------- | ---- | ---- | ---- | ------------- | ---------------------------- | -------------------- | ----------------------- |
| 리스트   | O    | O    | O    | `[]`          | `L[0]`, `in`                 | `append()`, `L[0]=V` | `del L[0]`              |
| 튜플     | O    | O    | X    | `()`          | `T[0]`, `in`                 | 불가                 | 불가                    |
| 딕셔너리 | O    | X    | O    | `{k: v}`      | `D[key]`, `D.get()`          | `D[key]=V`           | `del D[key]`            |
| 셋       | X    | X    | O    | `{}`, `set()` | `in` 가능 (인덱스 조회 불가) | `add()`              | `remove()`, `discard()` |

사례

1. 기본적인 경우(데이터관리, 순서 필요) -> 리스트
2. 검색(회원 정보, 설정 값) -> Dict
3. 절대 변경하면 안되는 값(좌표, 비밀번호, 상수 묶음) -> Tuple
4. 중복 제거(참가자 명단) -> Set
